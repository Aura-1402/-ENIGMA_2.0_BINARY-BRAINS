# Intelligent Interview Simulation System

## Problem
Students lack real interview experience and structured feedback.

## Solution
AI-powered system that simulates interviews, evaluates code, and analyzes explanations.

## Features
- AI interviewer
- Coding evaluation
- Voice analysis
- Resume-based questions

## Tech Stack
- Python
- Gemini API
- Flask
- HTML/CSS/JS

## How to run
Step 1: clone repo
Step 2: install requirements
Step 3: run main.py
